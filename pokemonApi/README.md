# Scripts for development

1. Clone the repository
2. Scripts execute

```
yarn install
```

3. have nest js installed

4. Docker run
```
docker-compose up -d
```

##Techologies used

* MongoDB
* Nest js
* Docker
* Mongoose
* Typescript
